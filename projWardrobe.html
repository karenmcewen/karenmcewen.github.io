Project: Wardrobe Model View Controller (MVC)

Overview: Create a web application for managing the contents of your wardrobe.
The app will allow you to add, edit, view, and delete items in your closet.
Skills Required: Database (SQL) and Model View Controller (MVC) as well as HTML, CSS, and Bootstrap

  • GitHub link

Choosing a "Wardrobe"

Doctor Who is a long-running British television series. Whenever a new actor takes on the role of Doctor, the character heads to the wardrobe aboard the ship (known as the TARDIS) to create a new look. I used my love of science fiction to create a usable database and website application to allow a user to create an outfit for a new Doctor. All photos are the property of the BBC.

EDMX file

The project entailed a series of 9 linked tables with multiple primary and foreign keys
Tables included: tops, bottoms, outerwear, neckwear, accessories, colors, seasons, occasions, and outfits.
